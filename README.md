# FragenMultipleChoice
Multiple-Choice extractor in HTML and JS

📱 Features
PDF-Import: Automatisches Extrahieren von Fragen im Format „Frage N:"
Antworten A–D: Antwort A wird immer als korrekt erkannt
Fortschritts-Tracking: Richtig beantwortete Fragen werden grün markiert und gespeichert
Zufällig-Modus: Konfigurierbare Anzahl zufälliger Fragen
Fragen-Browser: Kategorien, Suche, manuelle Auswahl
Einstellungen: Bereits korrekte überspringen, Reihenfolge mischen
Offline-fähig: Alle Daten lokal gespeichert (localStorage)
Responsive: Optimiert für iOS, Android und Desktop

📄 PDF-Format
Die App erwartet Fragen im Format:
Frage 1: Fragetext hier
A Richtige Antwort (immer A)
B Falsche Antwort
C Falsche Antwort
D Falsche Antwort
📦 Technologie

Vanilla HTML/CSS/JavaScript
PDF.js für PDF-Parsing (CDN, keine Installation nötig)
Keine Backend-Abhängigkeiten
